# Time delay face

This is the database related to paper '[ForgetMeNot: Memory-Aware Forensic Facial Sketch Matching](http://www.eecs.qmul.ac.uk/~tmh/papers/ouyang2016forgetmenot)' which is accepted by CVPR2016.
